# IM Accelerator Landing Page

A polished, responsive, frontend-only landing page built from scratch based on the reference masterclass page.
Link https://maria-ai-skills-test-3qiz.vercel.app/

## Files

- `index.html` — landing page structure
- `styles.css` — full responsive design and visual styling
- `script.js` — countdown timer, form validation, scroll reveal animations
- `assets/` — local SVG logo, illustrations, icon, and background pattern

## How to preview

Open `index.html` in any browser.

## How to edit

- Main headline/copy: edit `index.html`
- Colors, spacing, typography: edit `styles.css`
- Countdown date logic: edit `script.js`
- Form connection: connect the form submit handler in `script.js` to your CRM, webhook, or backend.

## Notes

The current form is frontend-only and displays a success message after validation. To collect leads, connect it to GoHighLevel, HubSpot, Mailchimp, ConvertKit, Zapier, Make, or a custom endpoint.
